# دليل استخدام Expo Go

## التغييرات التي تم تطبيقها

✅ تعطيل New Architecture (قد يسبب مشاكل مع Expo Go)
✅ إضافة bundleIdentifier لـ iOS
✅ إضافة أذونات الكاميرا ومعرض الصور
✅ إضافة plugins للكاميرا و image picker

## طرق تشغيل المشروع مع Expo Go

### الطريقة 1: استخدام Tunnel Mode (موصى بها)

هذه الطريقة تعمل حتى لو كان الجهازان على شبكات مختلفة:

```bash
npx expo start --tunnel
```

**ملاحظة**: قد تحتاج إلى تسجيل الدخول لحساب Expo:
```bash
npx expo login
```

### الطريقة 2: استخدام LAN Mode (الوضع الافتراضي)

```bash
npm run dev
```

**المتطلبات**:
- يجب أن يكون الكمبيوتر والآيفون على نفس شبكة WiFi
- تأكد من عدم تفعيل VPN
- تأكد من أن جدار الحماية لا يحجب الاتصال

### الطريقة 3: استخدام Local Mode

```bash
npx expo start --localhost
```

**استخدم هذه الطريقة إذا كنت تستخدم محاكي على نفس الجهاز**

## خطوات الاستخدام

1. **تشغيل الخادم**:
   ```bash
   npx expo start --tunnel
   ```

2. **انتظر حتى يظهر QR Code**

3. **على الآيفون**:
   - افتح تطبيق Expo Go
   - اضغط على "Scan QR code"
   - وجّه الكاميرا نحو QR Code الظاهر في الطرفية

4. **انتظر حتى يتم تحميل التطبيق**

## حل المشاكل الشائعة

### المشكلة: "Unable to connect"

**الحلول**:
1. استخدم Tunnel Mode:
   ```bash
   npx expo start --tunnel
   ```

2. تأكد من تحديث Expo Go:
   - اذهب إلى App Store
   - ابحث عن "Expo Go"
   - تأكد من تثبيت أحدث إصدار

3. أعد تشغيل الخادم:
   ```bash
   # اضغط Ctrl+C لإيقاف الخادم
   # ثم شغّله مرة أخرى
   npx expo start --tunnel
   ```

### المشكلة: "Network response timed out"

**الحلول**:
1. تأكد من اتصال الإنترنت
2. استخدم Tunnel Mode بدلاً من LAN Mode
3. تأكد من عدم تفعيل VPN على أي من الجهازين

### المشكلة: التطبيق لا يحمل أو يظهر شاشة بيضاء

**الحلول**:
1. امسح الـ cache:
   ```bash
   npx expo start --clear
   ```

2. أعد تثبيت dependencies:
   ```bash
   npm install
   ```

3. في Expo Go، اهز الجهاز وأعد تحميل التطبيق

## معلومات إضافية

### متى تستخدم كل طريقة؟

- **Tunnel**: عندما تواجه مشاكل في الاتصال، أو عند استخدام شبكات مختلفة
- **LAN**: للتطوير السريع عندما تكون على نفس الشبكة
- **Localhost**: عند استخدام محاكي على نفس الجهاز

### تحديث Expo Go

من المهم جداً أن يكون لديك أحدث إصدار من Expo Go:

1. افتح App Store على iPhone
2. ابحث عن "Expo Go"
3. إذا ظهر زر "تحديث"، اضغط عليه
4. انتظر حتى يكتمل التحديث

## البديل: Development Build

إذا استمرت المشاكل، يمكنك إنشاء Development Build:

```bash
# تثبيت EAS CLI
npm install -g eas-cli

# تسجيل الدخول
eas login

# إنشاء build
eas build --profile development --platform ios
```

**ملاحظة**: هذا يتطلب حساب Apple Developer ($99/سنة) لتثبيته على iPhone حقيقي.

3. امسح الـ cache
4. أعد تشغيل كل من الخادم والتطبيق
